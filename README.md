# Simple Apps

Small web apps collected in one repo, with a simple `index.html` landing page at the root for GitHub Pages.

Current apps:
- `Stickynote` at `stickynote/`

Structure:
- `index.html`: homepage with links to each app
- `stickynote/`: sticky note board app

Publishing idea:
- host the repo with GitHub Pages
- use the root `index.html` as the app directory
- let each app live in its own folder
